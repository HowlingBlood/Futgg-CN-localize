# FUT.GG & FUTBIN汉化插件

一键汉化FUT GG和FUTBIN插件，无需额外操作

## 前置要求

Tampermonkey 或 脚本猫

## 安装教程

- 脚本猫
  - 点击[链接](https://scriptcat.org/zh-CN/script-show-page/5533)，点击“安装脚本”，安装完成即可使用
- Tampermonkey
  - 点击[链接](https://greasyfork.org/zh-CN/scripts/568763)，点击“安装脚本”，安装完成即可使用

## 使用须知
网络不畅情况下可能出现没有汉化的情况，可点击右下角的刷新按钮
