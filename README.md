# FUT.GG 汉化插件

一键替换FUT GG汉化文本插件，无需额外操作

## 前置要求

Tampermonkey 或 脚本猫

## 安装教程

- 脚本猫
  - 点击[链接](https://scriptcat.org/zh-CN/script-show-page/5533)，点击“安装脚本”，安装完成即可使用
- Tampermonkey
  - 点击[链接](https://greasyfork.org/zh-CN/scripts/568763)，点击“安装脚本”，安装完成即可使用
- 手动安装
  - 复制futgg-logic.js里面的内容
  - 点击Tampermonkey / 脚本猫图标 -> 点击“添加新脚本” / 新建脚本 -> 删掉输入框内所有内容，复制粘贴futgg-logic.js里面的内容到输入框内 -> Ctrl+S保存
  - 打开FUT.GG

## 已知问题
首次访问FUTGG时可能会出现没有汉化的情况，多刷新几次即可
